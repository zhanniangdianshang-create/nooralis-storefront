# Nooralis Video Assets

Place authorized video files in this folder with these exact names:

- `commercial-application.mp4` - real commercial event or stage application footage.
- `product-demo.mp4` - product operation, setup, or flight demonstration footage.

Recommended export:

- Format: MP4 / H.264
- Resolution: 1080p or higher
- Aspect ratio: 16:9
- Duration: 15-45 seconds for landing page conversion
- Audio: optional, but avoid copyrighted music unless licensed

Only use videos you own, recorded yourself, received from the supplier with written permission, or licensed for commercial reuse.
