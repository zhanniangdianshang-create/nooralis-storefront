After-sales service

Submit a return or after-sales request.

Use this page for damaged-on-arrival cases, missing parts, wrong configuration, or other after-sales support. We verify the order first, then review the issue and contact the buyer with the next step.

If email reminders are active, the store inbox and the buyer email will both receive a confirmation after submission.

01

Verify the order

Enter the Nooralis order reference, matching buyer email, and PayPal transaction ID so we can match the request.

02

Describe the issue clearly

Choose the reason, item condition, and requested resolution so the service desk can triage the request faster.

03

Wait for manual review

Each request is reviewed before a return route, replacement plan, or support reply is given.
